Universal Tag Set

A set of 12 universal POS tags described in A Universal Part-of-Speech Tagset by Slav Petrov, Dipanjan Das and Ryan McDonald (2012) In Proceedings of the Eight International Conference on Language Resources and Evaluation (LREC'12) (code and mappings available: here.
